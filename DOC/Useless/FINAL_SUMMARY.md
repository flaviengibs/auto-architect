## 🎯 Commandes Disponibles

### Analyse Standard
```bash
npm run build              # Compiler
npm run analyze            # Analyser projet courant
npm run analyze:example    # Analyser exemple TypeScript
npm run analyze:python     # Analyser exemple Python
npm run analyze:self       # Auto-analyse
```

### Analyse Avancée
```bash
npm run analyze:security   # Avec détection sécurité
npm run analyze:trends     # Avec comparaison temporelle
npm run analyze:json       # Export JSON
npm run analyze:md         # Export Markdown
npm run analyze:html       # Export HTML
npm run diagram            # Générer diagramme
```

### Watch Mode
```bash
npm run watch              # Watch mode basique
npm run watch:threshold    # Watch avec seuil 80
```

### Personnalisées
```bash
# Analyse complète
node dist/cli/index.js analyze \
  --security \
  --compare previous.json \
  --format html \
  --output report.html \
  --diagram mermaid \
  --threshold 70

# Watch avec options
node dist/cli/index.js watch \
  --debounce 5000 \
  --threshold 80 \
  --format json
```
